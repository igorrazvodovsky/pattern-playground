# Nav bar

The provision of persistent, global access to an application's primary destinations, balancing reach against cognitive overhead.

<Story of={ NavBarStories.NavBar } />

Nav bars are the primary UI implementation of the [fully connected](../?path=/docs/actions-navigation-fully-connected--docs) navigation model, where every page links to all major sections through global navigation. See [information architecture](../?path=/docs/foundations-information-architecture--docs) foundation for the broader structural pattern.

Nav bars can also implement:
- [Multilevel tree](../?path=/docs/actions-navigation-multilevel-tree--docs) - when using dropdowns or expandable menus for hierarchical navigation
- Hybrid models - combining global access with contextual navigation
