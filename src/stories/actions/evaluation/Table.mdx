# Table

The structured comparison of items across shared dimensions, balancing information density with scanability.

## Advanced use cases

As a component, the table is a lightweight wrapper providing styles and basics like sticky headers. While this suits basic tables, more complex features like sorting and resizing are needed for advanced use cases. This kind of functionality is out of scope for the table component, as the spectrum of possible use-cases is too broad for a single component to cover effectively. However, this minimalist approach allows for seamless integration with external libraries that provide those advanced features.
