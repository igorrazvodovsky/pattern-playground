## AI-augmented form

Forms force users to translate their reality into database fields.
AI allows systems to handle the structure, accepting [forgiving input](../?path=/docs/actions-application-data-entry--docs#forgiving-format) from the user.

In an AI-augmented form, the primary user action shifts from data *entry to data *verification*.
1. Input: User provides raw data (image, PDF, pasted text, voice dump).
2. Extraction: AI maps this to the form schema.
3. Verification: User reviews the pre-filled form for accuracy.
