# Overflow

Governs how text and continuous content respond when they exceed their container. For discrete sets of UI elements — toolbars, tab bars, navigation items — see [priority+](../?path=/docs/actions-coordination-priority--docs).

## Decision tree

<MermaidDiagram chart={`
flowchart TD
    A[Content exceeds container?] --> B{Is content critical?}
    B -->|Yes| C{Can container expand?}
    B -->|No| D[Use overflow: hidden]
    C -->|Yes| E[Allow container to grow]
    C -->|No| F{Is content scrollable?}
    F -->|Yes| G[Use overflow: scroll/auto]
    F -->|No| H{Can content be truncated?}
    H -->|Yes| I[Use text overflow ellipsis]
    H -->|No| J[Consider redesign/priority]
`} />
