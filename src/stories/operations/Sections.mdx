# Sections

An adaptive container that presents the same content structure through different affordances depending on context.

<Story of={SectionsStories.Responsive} />

Content often divides into a small, stable set of named parts. Whether those parts appear as horizontal tabs or collapsible disclosure regions is not a property of the content — it's a property of the *context*: available space, information density, reading mode.
Sections treats this as a single design decision rather than a choice between separate components.

The two modes — tab bar and details — are not layout options but *disclosure affordances*: different ways of managing which content is visible and how the actor navigates between parts.
