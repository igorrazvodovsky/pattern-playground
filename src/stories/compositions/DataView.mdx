# Data view

Displays a [collection](../?path=/docs/concepts-collection--docs), enabling users to quickly scan, explore, and interact with [items](../?path=/docs/concepts-item--docs).
By combining varied representations with filtering, sorting, and grouping, data view provides tools for framing the data.

## Saved views

Saved views reduce friction for repeat tasks, support different user roles, and provide continuity across sessions.

- Developers define default views that align with the application's core use cases or intended user journeys
- Administrators configure role-specific or task-specific views shared across teams or departments
- End users create personalised views tuned to their workflow or preferences
