# Bubble menu

A contextual menu that appears when users select content, providing relevant actions for the selected text or elements. Enables users to act on specific content without interrupting their reading or editing flow.

## Variants

### Single action

Focused interaction for a primary action on selected content.

## Anatomy

- **Trigger**: Text or content selection
- **Container**: Floating menu positioned near selection
- **Actions**: Contextually relevant buttons or controls
- **Positioning**: Automatically positioned to avoid viewport edges

## Behaviour

- **Appears** when content is selected
- **Disappears** when selection is cleared or focus moves
- **Repositions** automatically to remain visible
- **Persists** during action execution

## States

- **Hidden**: Default state when no content is selected
- **Visible**: Appears with relevant actions for selected content
- **Loading**: Actions may show loading states during execution
- **Active**: Buttons show active state for applied formatting
